# Rithvik Malay Portfolio

This is a personal portfolio website built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

## Available Scripts

- `npm install` – install dependencies
- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally

## Tech Stack

- React + TypeScript
- Vite bundler
- Tailwind CSS
- shadcn/ui component library

You can customize the content in the `src` directory (sections, projects, experience, etc.) to keep the site up to date with your latest work.
